# Handwriting Go Away

Handwriting text generator. https://storage.yusanshi.com/handwritinggoaway/

## Get started

First download fonts into `public/fonts/`, modify value of `font` in `src/locales/[LOCALE]/main.json` accordingly.

```
npm install
npm run serve

# build
npm run build
```
